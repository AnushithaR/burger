# Mouth Controlled Food Catch Game

## Overview
Mouth Controlled Food Catch Game is an interactive computer vision project developed using Python, OpenCV, and MediaPipe. The game uses real-time webcam input to detect mouth movements.

## Features
- Real-time face and mouth tracking
- Webcam-based game control
- Random food item generation
- Score tracking system
- Interactive gameplay
- Uses Computer Vision concepts

## Technologies Used
- Python
- OpenCV
- MediaPipe
- NumPy

## How to Run
1. Install the required Python libraries.
2. Connect a webcam.
3. Run `food_game.py`.
4. Open your mouth to catch food items.
5. Avoid harmful objects.

## Project Structure
- `food_game.py` - Main game logic
- `camera_test.py` - Camera testing script
- `burger.png`, `pizza.png`, `broccoli.png`, `bomb.png` - Game assets
